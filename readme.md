一、什么是栅格系统
Bootstrap 提供了一套响应式、移动设备优先的流式栅格系统，随着屏幕或视口（viewport）尺寸的增加，系统会自动分为最多12列。包含了用于简单的布局选项的预定义类，也包含了用于生成更多语义布局的功能强大的混合类。
内容：决定什么是最重要的。
布局：优先设计更小的宽度。
渐进增强：随着屏幕大小增加而添加元素。
